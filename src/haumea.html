Haumea

Verbinden auf die Haumea Rechenknoten der Universität Rostock

Es muss eine aktive Verbindung zum Netzwerk der Universität Rostock bestehen.
  • Eduroam Netzwerkverbindung
  • Cisco-VPN:Link
Über Konsole per ssh verbinden:
  • ssh haumea1.hpc.uni-rostock.de -l [BENUTZER]
  • ssh haumea2.hpc.uni-rostock.de -l [BENUTZER]

Befehle auf dem Haumea Server

Neue Arbeitsumgebung angelegen.
  1. AllgemeineLinuxBefehle.
  2. Neue Umgebung anlegen:
    • python3 -m venv ~/envs/env
  3. Umgebung aktivieren:
    • source ~/envs/env/bin/activate
  4. pip-Packete über Proxy installieren:
    • pip install --proxy=http://proxy.uni-rostock.de:8080 "Packagepip "
Prozess starten und Überwachen.
  1. Linux.sh-Datei anlegen.
  2. Prozess Starten:
    • sbatch [NAME].sh
  3. Prozess überprüfen:
    • squeue -u [BENUTZERNAME]
  4. Auf GPU einloggen:
    • ssh gpu1 (gpu2,...,gpuX)
    • htop
  5. Aus GPU ausloggen:
    • Str+D/Control+D
