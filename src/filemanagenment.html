Dateienverwaltung

tar.gz

Packen
  • tar cfvz FILENAME.tar.gz FILENAME
Entpacken
  • tar -xvzf FILENAME.tar.gz
Options
Shortcut Funktion
-c create: neue tar.gz Datei erzeugen
-v verbose mode: alle Dateien werden beim packen angezeigt
-f eine Datei wird angegeben, die gepackt wird
-z tar mit gzip packen

Datenübertragung mit scp

Beispiel: Daten von Haumea auf PC.
  • scp -r jt292@haumea1.hpc.uni-rostock.de:/home/[BENUTZER]/[DATEIPFAD] /Users/[ZIELPFAD]
